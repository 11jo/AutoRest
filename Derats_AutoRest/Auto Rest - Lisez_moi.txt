Ce module très simple a été crée à la demande de DrAzTIK, qui souhaiter pouvoir récupérer ses sorts entre chaque combat.

Le module comporte deux options :
Comment appliquer le repos :
1 : En appuyant sur une touche (D, non configurable). Cela simule un repos dans le jeu
2 : Automatique après chaque combat; Cela simule les comportement des jeux récents comme Pillars of Eternity ou Dragon Age Origine
3 : Les deux. Parce qu'il n'y a pas toujours un combat pour déclencher le script de fin de combat.

Quel repos ?
1 : tous les sorts rendus + 3D8 pvs rendus~
2 : tous les sorts rendus + 3D8 pvs rendus + dissipation 100%
3 : tous les sorts rendus + Guérison
4 : tous les sorts rendus + Guérison + dissipation 100%
5 : tous les sorts rendus + Restauration majeure
6 : tous les sorts rendus + Restauration majeure + dissipation 100%
7 : tous les sorts rendus + Résurrection + Restauration majeure~
8 : tous les sorts rendus + Résurrection + Restauration majeure + dissipation 100%

Notes :
- L'appui sur la touche D ne marche pas si l'IA est désactivée. Le repos "automatique" est toujours actif.
- la guérison totale simule l'option légale "repos jusqu'à guérison".
- la dissipation 100% simule la dissipation des buffs après repos.
- la simulation du repos standard est "3D8 pvs rendus ou guérison, + dissipation, en appuyant sur la touche D". Toutes les autres combinaisons sont de la triche. (Et encore, l'appui sur la touche D ne s’encombre pas de vérifier si des ennemis sont présents ou si la zone n'autorise pas le repos).
- Si vous voulez modifier la touche nécessaire pour activer le repos, il vous suffit d’éditer le fichier "drazres2.baf" (avant d'installer) et de changer la ligne "HotKey(D)" par "HotKey(la touche que vous voulez)"
